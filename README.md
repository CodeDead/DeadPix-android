# DeadPix
DeadPix for Android!

This app can be used to locate and fix dead or stuck pixels on your screen. No advertisements, no tracking code, no special permissions required!

# Requirements
* Android 28+
* Android Studio

## Translations
- Portuguese (Brazil) - [SnwMds](https://github.com/SnwMds)
